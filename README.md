# Vue 3 + Vite To-Do List App Using Local Storage

Tutorial Found [Here] (https://www.youtube.com/watch?v=qhjxAP1hFuI)

## Relevant Commands to Run App
- Open main folder
- Ensure you have React JS installed
- Ensure you have Vue JS installed
```
 npm install
 npm run dev
```

This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.
